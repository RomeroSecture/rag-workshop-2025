# 📁 Nivel 1 - Básico

Guarda aquí tus **primeras implementaciones** de los ejercicios del workshop.

## 🎯 Objetivo

Este nivel es para:
- ✅ Soluciones funcionales pero simples
- ✅ Código que "funciona" aunque no sea perfecto
- ✅ Primeras iteraciones de los ejercicios

## 📝 Ejemplo de uso

Cuando completes un ejercicio del **Módulo 2**, puedes guardar tu solución aquí:

```python
# 02_mi_ejercicio1.py
# Mi primera solución al ejercicio de optimización de chunk_size

chunk_sizes = [300, 500, 800, 1000, 1500]
# ... tu código aquí ...
```

## 💡 No te preocupes por la perfección

En este nivel:
- ❌ No necesitas código perfecto
- ❌ No necesitas manejo de errores completo
- ❌ No necesitas documentación exhaustiva

Solo necesitas:
- ✅ Que funcione
- ✅ Que entiendas lo que hace
- ✅ Que resuelva el ejercicio

#### 6. TODO 4: Streaming con Server-Sent Events (8 min) ‚≠ê **FEATURE CR√çTICA**

**Instructor explica:**
> "Streaming permite mostrar la respuesta mientras se genera, como ChatGPT. Mejora UX dram√°ticamente."

**Implementaci√≥n:**
```python
from fastapi.responses import StreamingResponse

@app.post("/query/stream")
async def query_stream(request: QueryRequest):
    """Query con streaming"""
    async def generate():
        # Yield chunks a medida que se generan
        for chunk in rag_system.query_stream(request.question):
            yield f"data: {json.dumps(chunk)}\n\n"

    return StreamingResponse(
        generate(),
        media_type="text/event-stream"
    )
```

**Demo en vivo:**
